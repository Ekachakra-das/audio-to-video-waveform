# Waveform Web Tool (GitHub Pages)

Этот проект позволяет создавать видео с визуализацией звуковой волны (столбиками) прямо в браузере, используя ресурсы вашего компьютера.

## Особенности
- **Премиальный дизайн**: Волна рисуется в виде аккуратных столбиков (Bars).
- **FFmpeg.wasm**: Обработка видео происходит локально в браузере с помощью WebAssembly.
- **Приватность**: Ваши аудиофайлы не отправляются на сервер.
- **GitHub Pages Ready**: Включен скрипт `coi-serviceworker.js` для обхода ограничений безопасности (Cross-Origin Isolation).

## Как запустить локально
Просто открыть `index.html` двойным кликом не получится из-за ограничений безопасности браузера. Нужно запустить локальный веб-сервер:

1. Откройте терминал в этой папке.
2. Запустите сервер (если есть Python):
   ```bash
   python3 -m http.server 8000
   ```
3. Откройте в браузере: `http://localhost:8000`

## Как развернуть на GitHub Pages
1. Создайте новый публичный репозиторий.
2. Загрузите файлы: `index.html`, `coi-serviceworker.js`.
3. Перейдите в **Settings -> Pages** и включите GitHub Pages для вашей ветки.
4. Обязательно убедитесь, что файлы лежат в корне или правильно указаны пути. Скрипт `coi-serviceworker.js` сам настроит нужные заголовки для работы FFmpeg.

## Структура файлов
- `index.html` — основной интерфейс и логика.
- `coi-serviceworker.js` — магия для включения Cross-Origin Isolation (нужно для FFmpeg.wasm).
